# AI & Developer Job Market Analysis Dashboard

## Project Overview

This project analyzes job market trends using Power BI. The dashboard provides insights into job roles, salaries, AI adoption, work modes, and in-demand programming skills based on a manually collected dataset from job postings.

## Objectives

* Analyze job opportunities across different cities.
* Compare Full-Time and Internship openings.
* Identify the most in-demand programming languages.
* Analyze AI tool adoption in job roles.
* Study the relationship between experience and salary.
* Compare salaries for AI-related and non-AI-related jobs.

## Tools Used

* Power BI
* Power Query
* Microsoft Excel

## Dataset

The dataset was created from job postings collected from platforms such as LinkedIn, Naukri, and Indeed.

### Dataset Fields

* Job Title
* Job Role
* City
* Experience Required
* Programming Language
* Work Mode
* Job Type
* Salary (LPA)
* AI Tools Used

Summary of major insights derived from the analysis.

## Key Insights

* Python emerged as one of the most in-demand programming languages.
* AI-related jobs generally offered higher salaries.
* Experience showed a positive correlation with salary.
* Remote and Hybrid work modes accounted for a significant share of job postings.
* AI tools such as ChatGPT and GitHub Copilot appeared frequently in job requirements.
